# Gould-Street-University-Portal-SLAP [(Video Demo)](https://drive.google.com/drive/folders/1sQtSXmm1z4RC78WRkljfrWyqBEEXfhci?usp=drive_link)

Gould Street University SLAP is a full-featured university portal system designed to simulate a real academic learning platform similar to D2L (Brightspace). It supports assignment submissions, grading, course management, and administrative oversight. The system was developed as part of a team software engineering project and demonstrates role-based access control and collaborative academic workflows.

---

## 📌 Overview

The SLAP (Student Learning and Academic Portal) system provides a centralized platform for managing university activities, including:

- Assignment submission and tracking  
- Grading and feedback distribution  
- Course enrollment and dashboard management  
- Instructor evaluation workflows  
- Administrative student and course management  

It simulates real-world academic systems used in universities and demonstrates practical full-stack software engineering principles.

---

## 👥 User Roles

### 🎓 Students
- View enrolled courses  
- Submit assignments and projects  
- Receive grades and feedback  
- Track academic progress through dashboard  

### 👨‍🏫 Instructors
- Create and manage courses  
- Upload assignments and instructions  
- Grade student submissions  
- Provide feedback and evaluations  

### 🛠️ Administrators
- Manage student accounts  
- Assign courses to students  
- Review submissions and grading activity  
- Oversee system-wide academic data
